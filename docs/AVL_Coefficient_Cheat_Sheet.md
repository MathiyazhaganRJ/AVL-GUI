# AVL Stability Output (.st): The Engineer's Reading Chart

When you execute the `st` command in the `.OPER` menu, AVL outputs a massive wall of coefficients. This cheat sheet translates every single variable into its physical flight dynamics meaning.

---

## How AVL Calculates These Numbers (The Mathematics)
When you ask AVL to run a stability analysis, it performs a **Small Perturbation (Finite Difference) Analysis**. Here is exactly what happens under the hood:

1.  **The Base State:** AVL calculates the baseline aerodynamic forces for the exact flight condition you specified (e.g., Alpha = 10 deg).
2.  **The Matrix Nudges:** It then mathematically "nudges" the aircraft by an infinitesimal amount in every single axis, one at a time. It pitches the plane up a fraction of a degree, adds a tiny sideslip, adds a tiny mathematical rotation (p, q, r), and slightly deflects every control surface.
3.  **The Gradients:** It recalculates the forces for each "nudged" state, finds the difference from the base state, and divides by the nudge amount (e.g., Change in C_m / Change in Alpha). 

The resulting slopes/gradients are the coefficients printed in the terminal. Because AVL uses a linearized Vortex Lattice Method (VLM), it can calculate this massive mathematical matrix in milliseconds.

## Aerodynamic Coefficients Nomenclature
Aerodynamic variables in AVL follow a strict mathematical naming convention. The variable name is constructed by combining a primary force/moment identifier (the prefix) with its kinematic dependency (the suffix).

### 1. The Prefix (The Dependent Variable)
*   **Capital Letters** (`C_L`, `C_Y`, `C_D`) represent **Forces** (Lift, Side Force, Drag).
*   **Lowercase Letters** (`C_l`, `C_m`, `C_n`) represent **Moments** (Rolling, Pitching, Yawing).

### 2. The Suffix (The Independent Variable)
The trailing letter designates the specific flight condition or kinematic rate driving the change:
*   `a` : Alpha (Angle of Attack)
*   `b` : Beta (Sideslip Angle)
*   `p` : Roll Rate
*   `q` : Pitch Rate
*   `r` : Yaw Rate
*   `d` : Control Surface Deflection

**Applied Examples:**
*   **`CLp`** : `C_L_p` -> The change in total **L**ift force due to **p** (roll rate).
*   **`Clp`** : `C_l_p` -> The change in rolling **l**ing moment due to **p** (roll rate). Known as *Roll Damping*.
*   **`Cmq`** : `C_m_q` -> The change in pitching **m**oment due to **q** (pitch rate). Known as *Pitch Damping*.
*   **`Cnb`** : `C_n_b` -> The change in yawing **n**oment due to **b** (beta/sideslip). Known as *Directional Stability*.

## 1. Total Aerodynamic Forces & Moments (Trim State)
These coefficients represent the overall forces acting on the aircraft at your current trimmed flight condition.

| Coefficient | Name | Description | Target / Rule |
| :--- | :--- | :--- | :--- |
| **CLtot** (or `CL`) | Total Lift Coefficient | Total aerodynamic lift generated by the aircraft. | Depends on target cruise weight/speed. |
| **CDtot** (or `CD`) | Total Drag Coefficient | Total aerodynamic drag (Induced + Profile). | Minimize. |
| **Cmtot** (or `Cm`) | Total Pitching Moment | The tendency of the aircraft's nose to pitch up or down. | **0.0** (if perfectly trimmed for level flight). |
| **Cltot** (or `Cl`) | Body-Axis Rolling Moment | Roll tendency relative to the physical fuselage. | **0.0** (unless flying asymmetrically). |
| **Cntot** (or `Cn`) | Body-Axis Yawing Moment | Yaw tendency relative to the physical fuselage. | **0.0** (unless in a slip). |
| **Cl'** | Stability-Axis Roll | Roll tendency relative to the *wind direction* (accounts for high Alpha). | **0.0** |
| **Cn'** | Stability-Axis Yaw | Yaw tendency relative to the *wind direction* (accounts for high Alpha). | **0.0** |
| **CX, CY, CZ** | Body-Axis Forces | Raw forces purely relative to the physical fuselage XYZ axes. | Used for structural/inertial analysis. |

## 2. Efficiency & Trefftz Plane (Wing Performance)
These variables tell you how efficiently your wing geometry generates lift.

| Coefficient | Name | Description | Target / Rule |
| :--- | :--- | :--- | :--- |
| **CDind** (or `CD_i`) | Induced Drag Coefficient | Drag created purely by the production of lift (wingtip vortices). | Minimize via twist/taper optimization. |
| **CDvis** (or `CD_p`) | Profile Drag Coefficient | Parasitic/Viscous drag (skin friction). Assumes 0.0 unless XFOIL `.pol` files are loaded. | Minimize via airfoil selection. |
| **CLff** | Trefftz Plane Lift | Total lift integrated at the Trefftz plane (wake) far downstream. | Should closely match `CLtot`. |
| **CDff** | Trefftz Plane Drag | Total induced drag integrated at the Trefftz plane. Highly accurate. | Should closely match `CDind`. |
| **e** | Oswald Efficiency | A measure of how close your spanwise lift distribution is to a mathematically perfect ellipse. | **> 0.85**. Physically CANNOT exceed 1.0. If e > 1.0, you forgot `YDUPLICATE` or have overlapping geometry. |

---

## 3. Stability-Axis Derivatives (Static Stability)
These derivatives define how the aircraft reacts to wind gusts from the front (`a` / Alpha) or the side (`b` / Beta).

| Coefficient | Name | Description | Target / Rule |
| :--- | :--- | :--- | :--- |
| **CLa** | Lift Curve Slope | How rapidly lift increases as angle of attack increases. | High for performance, low for gust tolerance. |
| **Cma** | Pitch Stability | **CRITICAL:** How the pitching moment changes with Alpha. | **< 0 (Negative).** Must point nose down when gust hits. |
| **CDa** | Drag vs. Alpha | How drag changes as angle of attack increases. | Always positive. |
| **CYa** | Side Force vs. Alpha | Usually 0 for symmetric aircraft. | 0.0 |
| **Cla** | Roll vs. Alpha | Usually 0 for symmetric aircraft. | 0.0 |
| **Cna** | Yaw vs. Alpha | Usually 0 for symmetric aircraft. | 0.0 |
| **CLb** | Lift vs. Beta | Usually 0 for symmetric aircraft. | 0.0 |
| **CYb** | Side Force vs. Beta | How much side force is generated when flying in a slip. | Negative (wind pushes fuselage sideways). |
| **Cmb** | Pitch vs. Beta | Usually 0 for symmetric aircraft. | 0.0 |
| **Cnb** | Weather-cocking | **CRITICAL:** Directional stability. Vertical tail authority. | **> 0 (Positive).** Nose must turn into the wind. |
| **Clb** | Dihedral Effect | **CRITICAL:** Lateral stability. Wing dihedral authority. | **< 0 (Negative).** Must roll away from the sideslip. |
| **Xnp** | Neutral Point | The aerodynamic center of the whole aircraft. | CG must be **in front** of Xnp (5-15% margin). |
| **Spiral** | Clb Cnr / Clr Cnb | The spiral stability index. | **> 1.0** for a naturally spirally stable aircraft. |

---

## 4. Rotary Derivatives (Dynamic Damping)
These tell you how the aircraft behaves while it is actively rotating (Pitch rate `q`, Roll rate `p`, Yaw rate `r`).

| Coefficient | Name | Description | Target / Rule |
| :--- | :--- | :--- | :--- |
| **CLq** | Lift due to Pitch Rate | Transient lift generated by the tail moving downward during pitch-up. | Positive. |
| **Cmq** | Pitch Damping | **CRITICAL:** Resistance to pitching rotation. | **Heavily Negative.** Acts as a pitch shock absorber. |
| **Clp** | Roll Damping | **CRITICAL:** Resistance to rolling rotation. | **Heavily Negative.** Stops infinite roll acceleration. |
| **Cnp** | Adverse Yaw | Tendency to yaw in the opposite direction of a roll. | Negative. Requires rudder coordination. |
| **CYr** | Side Force due to Yaw | Force generated by the vertical tail swinging during a yaw. | Positive. |
| **Clr** | Roll due to Yaw | Outer wing moves faster in a yaw, creating more lift and rolling. | Positive. |
| **Cnr** | Yaw Damping | **CRITICAL:** Resistance to yawing rotation. | **Heavily Negative.** Stops the plane from spinning like a frisbee. |
| **CDq, CDp, CDr** | Drag due to Rotation | Minor drag increases due to rotational velocities. | Small magnitude. |

---

## 5. Control Derivatives (Servo Authority & Cross-Coupling)
These represent the effectiveness of your control surfaces and the secondary side-effects (cross-coupling) they produce when actuated. Note: `d01`, `d02`, etc., correspond to the index of the control surface defined in your `.avl` file.

### Primary Elevator Actuation (Pitch Control)
| Coefficient | Name | Description |
| :--- | :--- | :--- |
| **Cmd** | Primary Pitch Authority | The primary moment generated by the elevator. High magnitude required for rotation and landing flare. |
| **CLd** | Lift due to Elevator | **Secondary Effect:** Deflecting the elevator also changes the total lift of the aircraft slightly. |
| **CDffd** | Drag due to Elevator | **Secondary Effect:** Trim drag caused by deflecting the elevator. |

### Primary Aileron Actuation (Roll Control)
| Coefficient | Name | Description |
| :--- | :--- | :--- |
| **Cld** | Primary Roll Authority | The primary rolling moment generated by the ailerons. Determines maximum roll rate. |
| **Cnd** | Adverse Yaw | **Secondary Effect (CRITICAL):** Ailerons generate asymmetric drag, causing the nose to yaw away from the turn. Must be countered by the rudder or aileron differential. |

### Primary Rudder Actuation (Yaw Control)
| Coefficient | Name | Description |
| :--- | :--- | :--- |
| **Cnd** | Primary Yaw Authority | The primary yawing moment generated by the rudder. Must be large enough to counter adverse yaw (Cnd from ailerons) or engine failures. |
| **Cld** | Roll due to Rudder | **Secondary Effect:** The rudder creates a side force high above the CG, which induces a slight rolling moment when kicked. |
| **CYd** | Side Force due to Rudder | **Secondary Effect:** The pure lateral force generated by the rudder acting as a vertical wing. |

### Flap Actuation
| Coefficient | Name | Description |
| :--- | :--- | :--- |
| **CLd** | Lift due to Flaps | **Primary Effect:** Massive increase in lift coefficient for takeoff and landing. |
| **Cmd** | Pitching Moment | **Secondary Effect:** Deploying flaps usually causes a strong nose-down pitching moment (Cmd < 0) that must be countered by the elevator. |
| **CDffd**| Drag due to Flaps | **Secondary Effect:** Massive increase in induced and profile drag. |
